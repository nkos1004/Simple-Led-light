# Simple-Led-light
* simple led light that is controlled by an esp32
**
**A simple LED light controlled by an ESP32 microcontroller. This beginner-friendly project demonstrates basic GPIO control by turning an LED on and off using code uploaded to the ESP32.

**Features

**ESP32-controlled LED

**Easy to build and understand

**Great for learning electronics and programming

**Expandable for smart lighting and IoT projects

*Tech Used

**ESP32

**Arduino IDE

**C++
